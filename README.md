# Auth UI + Validation (HTML, CSS, JavaScript)

A clean Authentication UI (Login / Sign Up / Forgot Password) with strong client-side validation and a working demo user system using `localStorage`.

> ⚠️ Note: This project is **front-end only**. Password hashing here is **NOT secure** (demo purpose). Real authentication requires a backend.

## Features
- Login, Sign Up, Forgot Password screens (tab switch)
- Client-side validation:
  - Email format validation
  - Strong password rules (8+ chars, uppercase, lowercase, number, symbol)
  - Confirm password match
- Password strength meter
- Show/Hide password toggle
- Stores users in `localStorage` (so signup + login actually works)
- Reset demo data button

## Tech
- HTML
- CSS
- Vanilla JavaScript
- Browser localStorage

## How to Run
Just open `index.html` in your browser.

### Optional: Run with a local server (recommended)
If you have VS Code:
1. Install **Live Server** extension
2. Right click `index.html` → **Open with Live Server**
